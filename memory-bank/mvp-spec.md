# MVP 명세 (Wispr Flow Clone)

작성일: 2026-06-14

## 범위 한 줄 요약
맥에서 동작하는 데스크톱 앱. 단축키를 **누르고 있는 동안** 녹음하고, **떼면** 녹음 종료 → STT로 텍스트 변환 → 현재 커서 위치에 입력. 변환 기록은 앱 내 리스트에 날짜와 함께 내림차순으로 저장.

## 기능 요구사항 (필수)

### 1. 맥 데스크톱 앱
- macOS에서 동작하는 네이티브 데스크톱 앱
- 메뉴바(상태바) 앱 + 메인 윈도우 형태 권장

### 2. 단축키 녹음 (push-to-talk)
- 전역 단축키(global hotkey)를 **누르면 녹음 시작**
- 단축키를 **떼면 녹음 종료** (hold-to-record / push-to-talk 방식)
- 앱이 백그라운드에 있어도 단축키가 동작해야 함

### 3. 음성 → 텍스트 변환 (STT)
- 녹음한 오디오를 텍스트로 변환
- 변환 완료 후 결과 텍스트 확보

### 4. 커서에 텍스트 입력
- 변환된 텍스트를 현재 포커스된 앱의 커서 위치에 자동 입력
- (구현 방식: 클립보드 복사 후 Cmd+V 시뮬레이션, 또는 Accessibility API로 직접 삽입)

### 5. 변환 기록 리스트
- 앱 안에 변환된 텍스트 목록이 있어야 함
- 각 항목: **날짜(시각)** + **변환된 텍스트**
- 정렬: 최신순(날짜 **내림차순**)
- 로컬에 영구 저장(앱 재실행 후에도 유지)

### 6. 단축키 설정
- 사용자가 녹음 단축키를 직접 설정/변경할 수 있어야 함
- 설정한 단축키가 즉시 적용되고 저장됨

## MVP 제외(나중에)
- 실시간 스트리밍 받아쓰기(녹음 중 실시간 표시)
- 필러("음/어") 제거, 자동 포맷팅 등 AI 후처리
- 다국어 자동 감지, Whisper mode, 모바일/윈도우 지원
- 계정/로그인/결제

## 수락 기준 (Acceptance Criteria)
1. 앱 실행 시 메뉴바 아이콘 + 메인 윈도우(기록 리스트) 표시
2. 설정한 단축키를 누르고 있는 동안 녹음, 떼면 종료
3. 종료 직후 텍스트로 변환되어 현재 커서 위치에 입력됨
4. 변환된 내용이 리스트 최상단에 (날짜, 텍스트)로 추가됨
5. 앱을 껐다 켜도 리스트가 유지됨
6. 설정 화면에서 단축키를 바꾸면 새 단축키로 동작함

## 기술적으로 까다로운 지점 (미리 인지)
- **push-to-talk(키 떼면 종료)**: keyDown/keyUp 전역 감지 필요. Electron `globalShortcut`은 keyup 미지원 → 네이티브 이벤트 모니터 필요.
- **커서 삽입 & 전역 키 감지**: macOS **손쉬운 사용(Accessibility) 권한** 필요. 마이크 권한도 필요.
- STT 엔진 선택은 `project-architecture.md` 참고.
